Saving Account

You can add multiple type of saving account schemes. Each time, The software itself provides the suggested rate of interest you should write.

Saving Plus Account

It is hybrid type of account contains properties of both saving account & fixed deposit. Feature is coming soon.

Fixed Deposit Account

You can add multiple types of fixed deposit schemes. Each time, The software itself provides the suggested rate of interest you should write.

Recurring Deposit Account

You can add multiple types of recurring deposit schemes. Each time, The software itself provides the suggested rate of interest you should apply.

Loan Against Deposits

You can add multiple types of LAD (Loan against recurring deposit) schemes. Each time, The software itself provides the suggested rate of interest you should write. While disbursement, you cannot disburse the loan beyond due date of maturity.

General Loan – Type 1

You can add multiple types of loan account schemes, Where you have knowledge of loan disbursed amount & EMI amount only. Each time, The software itself calculate rate of interest as well as provides the suggested rate of interest you should charge.

General Loan – Type 2

You can add multiple types of loan account schemes, Where you have knowledge of loan disbursed amount & ROI only. Each time, The software itself calculates EMI as well as provides the suggested rate of interest you should charge.

Unencumbered Deposit

As per Companies Act, Every Nidhi company needs to maintain 10% of deposits held in last day of 2nd preceding month. Software is itself capable of letting you know the amount to make proper compliance.

User Right Control (URC)

Each important step in business can be properly monitored with the help of URC; Each person’s responsibility can be assigned & managed.

Collection Management

Daily due collection reports are tightly connected with the collection module. It has capacity of working on auto mode & customised mode both. Collection responsibility can be supervised technically.
